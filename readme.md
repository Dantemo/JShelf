# JShelf


## Pre-Install
Download project code and run:
```
npm install
```

## Usage

### Front-End
#### Development
```
gulp
```
or
```
gulp --build
```

#### Release
```
gulp --release
```

### Server-Side & Front End
#### Development
```
node server.js
```

#### Release
```
node server.js --release
```
